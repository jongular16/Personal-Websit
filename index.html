AI-Powered Detection of Dental Conditions

This project proposes an automated detection pipeline for analyzing dental panoramic X-rays using a YOLOv8 object detection model trained on a dataset of 9,000+ annotated images. My team and I implemented the interface, preprocessing, and postprocessing components, and integrated the model into a complete Flask-based web application. The system produces annotated images and condition summaries, showcasing experience in ML engineering, model evaluation, and real-world inference integration.
